Why this view exists

The existing ProfitKey · Work Order & Labor ERD shows the shape of ProfitKey's production data — the four tables (WORK_ORDER_MASTER, JOB_HOURS_SUMMARY, LABOR_TICKETS, EMPLOYEE_MASTER) and how they connect. This page shows the contents: ten read-only views into one historical shop order, so a quoter or cost analyst can see everything the record holds without opening ProfitKey itself.

Every view is anchored to Shop Order 0271926 — a closed work order for part P&H1J262CUS (“INT GEAR, 2ND RED RING 87T”, revision 21), released 3/2/2024 with a due date of 4/15/2024. Numeric values come verbatim from the Work Order Performance Detail Report for this order (User ID AALI, PKI date 5/17/2024).

This viewer is read-only. No records are created, duplicated, or edited — it surfaces what is already in ProfitKey for easier review. Phase A (this page) is the full-record view for one shop order. Phase B (coming) is the search/lookup experience across ProfitKey's work-order history.
