How These Initiatives Connect to the Current Project

Both initiatives are natural extensions of the data infrastructure proposed for this engagement. The Centralized Staging Database and Web Quoting Application would give OCG a clean, connected foundation — Case Management could plug into that foundation through Pipedrive, creating a single thread from first customer contact through project delivery and invoicing. Rather than a separate silo, it would become another node in the same connected system.

The AI Knowledge Capture initiative is arguably the most strategic of the two. The data analysis work scoped for this engagement — mapping where information lives across M1, ProfitKey, and the quoting system — would directly inform what a knowledge capture program should target first. WBD is well-positioned to lead this effort, using AI to extract, structure, and surface the best practices of OCG’s senior engineering and estimating team before those individuals retire.
