How the Systems Connect

The Staging Database is the linchpin — it holds mapped IDs that tie a Pipedrive deal to its M1 quote to its ProfitKey work order, so actual shop hours can be compared against the original estimate on every single job. Nightly Data Pipelines replicate critical tables from both on-premise ERPs read-only, leaving production systems untouched and the shop floor unaffected. The Web Quoting Application accesses the staging database through a secured API layer, which means estimators get full access to historical data and live actuals from any browser — without exposing the on-premise network.

Two new capabilities are built into the quoting workflow: Request For Materials (RFM) lets estimators solicit steel and metal pricing from suppliers directly within the quoting interface, rounding out the full cost picture before a quote is finalized. Tolerance Flagging automatically surfaces when the AI detects high-precision specs on a gear drawing — prompting the estimator early, since tight-tolerance jobs require specialized production routing and carry different cost profiles.
